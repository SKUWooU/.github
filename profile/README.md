## 프로젝트 개요
- 현재 많은 서비스들이 대용량 데이터를 처리하는 기술을 활용하고 있습니다. 이러한 트렌드를 반영하여, Spring Batch를 이용하여 KOPIS의 공공데이터를 주기적으로 크롤링하고 DB를 업데이트 하는 시스템을 구축했습니다.
- 대규모 트래픽이 발생할 경우 부하분산과 성능 향상을 위해서 웹 어플리케이션 서버를 이중화 구축했습니다.
